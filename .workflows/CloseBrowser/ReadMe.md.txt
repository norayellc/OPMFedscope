Input argument in_BrowserType should be one of these browser types : BrowserType.IE, BrowserType.Chrome, BrowserType.Edge, BrowserType.Firefox
Input argument in_BrowserObject is the browser object to be close. If the BrowserObject is provided the browsertype will not be considered
Input argument in_ForceKill if True the bot will kill the process based on info from (in_BrowserType or in_BrowserObject)
